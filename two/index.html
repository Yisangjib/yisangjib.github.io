매일 바닥에 떨어진 머리카락과 먼지를 줍고 글을 적을 때 쉼표의 위치와 물음표의 유무를 헤아리는 마음가짐으로, 채우기 텍스트처럼 의미없어 보이는 웹사이트 링크를 줍다가 문장 부호를 본다.

“첫째로 문장 부호는 말의 호흡, 억양 등을 위한 지침(index)이다. (…) 방점은 말의 시작에 들어가는 힘, 화제 전환 시 들이는 뜸, 힘주어 말하기 등과 같은 구술성을 함축한 문장부호의 기원이다. 둘째로, 낭독용이 아닌 보기용 상징 기호(symbol)이다. 조형적 상징에 기반한 특수문자는 구술성을 지닌 구두점과는 성격이 다르다.”
『마이크로 타이포그래피: 문장부호와 숫자』(워크룸프레스, 2015) 중 144페이지

. 마침표

U+002E


문장 부호의 하나. ‘.’의 이름이다. 서술ㆍ명령ㆍ청유 따위를 나타내는 문장의 끝에 쓰거나, 아라비아 숫자로 특정한 의미가 있는 날을 표시할 때, 장, 절, 항 등을 표시하는 문자나 숫자 다음에 쓴다. ≒끝점, 온점.

웹사이트의 도메인에 쓰이는 루트(root)를 표현한다. 예를 들어 'colonslashslash'와 'neocities'와 'org'를 구분시키는 역할을 하며, 보통 탑 레벨(org) 뒤에는 생략한다.

, 쉼표

U+002C


문장 부호의 하나. ‘,’의 이름이다. 같은 자격의 어구를 연결할 때 쓰거나, 짝을 지어 구별할 때, 이웃하는 수를 개략적으로 나타낼 때, 열거의 순서를 나타낼 때, 문장의 연결 관계를 분명히 하고자 할 때, 되풀이되는 말을 피하기 위해 일정 부분을 줄여서 열거할 때, 부르거나 대답하는 말 뒤에, 한 문장 안에서 ‘곧’ 따위의 어구로 다시 설명할 때, 한 문장에 같은 의미의 어구가 반복될 때, 도치된 어구 사이에, 바로 다음 말과 직접적인 관계에 있지 않음을 나타낼 때, 문장 중간에 끼어든 어구의 앞뒤에, 특별한 효과를 위해 끊어 읽는 곳을 나타낼 때, 짧게 더듬는 말을 표시할 때 쓴다. ≒반점, 콤마.

웹사이트 URL을 분리시킨다. URL에 쉼표가 있는 경우, 시스템에서 별도의 URL로 해석할 수 있다. 보통은 이미지 링크에 들어간다.

? 물음표

U+003F


문장 부호의 하나. ‘?’의 이름이다. 의문문이나 의문을 나타내는 어구의 끝에 쓰거나, 특정한 어구의 내용에 대하여 의심, 빈정거림을 표시할 때, 적절한 말을 쓰기 어려울 때, 모르거나 불확실한 내용임을 나타낼 때에 쓴다. ≒문표, 의문부, 의문 부호, 의문표.

쿼리 스트링(Query String)의 내용을 위해 사용한다. URL에 URL 매개변수를 삽입하여 클릭에 대한 정보를 URL이 추적하도록 하는데, 첫 번째 매개변수는 항상 물음표 뒤에 온다. 웹사이트에서 서버에 각종 정보를 넘겨주기 위한 시작점으로, 이 웹사이트 URL에서는 볼 수 없다. 굳이 넣어본다면, colonslashslash.neocities.org/? 변수값이 없으므로 같은 페이지다.

! 느낌표

U+0021


문장 부호의 하나. ‘!’의 이름이다. 감탄문이나 감탄사의 끝에 쓰거나, 어구, 평서문, 명령문, 청유문에 특별히 강한 느낌을 나타낼 때, 물음의 말로 놀람이나 항의의 뜻을 나타낼 때, 감정을 넣어 대답하거나 다른 사람을 부를 때 쓴다. ≒감탄부, 감탄 부호.

아스키 문자에 포함되어 무리없이 사용할 수 있다. 하지만 프로그래밍 언어에서 느낌표는 논리 부정 연산자로 쓰인다.

/ 빗금

U+002F


문장 부호의 하나. ‘/’의 이름이다. 대비되는 두 개 이상의 어구를 묶어 나타낼 때 쓰거나, 기준 단위당 수량을 표시할 때, 시의 행이 바뀌는 부분임을 나타낼 때 쓴다.

슬래시. 경로명에서 디렉터리 및 파일 이름을 구분 짓는 데에 사용한다. 슬래시 뒤에 경로가 붙지 않으면 colonslashslash.neocities.org///////////////////////////////////// 아무 의미가 없다.

= 쌍줄표

U+A4FF


인쇄에서, 두 항목이 같음을 나타내는 부호. ‘=’의 이름이다.

프로그래밍 언어에서 대입 연산자로 쓰인다. 좌변과 우변이 같다는 뜻이 아니다.

+ 더하기표

U+002B


덧셈의 부호 ‘+’를 이르는 말. ≒더하기표, 덧셈표, 보탬표, 플러스.

플러스. URL에서 플러스는 공백으로 대체한다.

- 빼기표

U+2212


뺄셈의 부호 ‘-’를 이르는 말.=뺄셈 부호.

하이픈. 단어와 단어 사이를 연결시킬 때 띄어쓰기 대신 사용한다.

# 번호 기호

U+0023


해시 기호(hash 記號), 번호 기호(番號 記號)는 기호의 일종으로, 전화기나 컴퓨터 자판에서 볼 수 있다. 기호의 모양이 올림표나 한자 "우물 정"(井) 자와 비슷하여 대한민국에서는 샵이나 우물정자로도 불리며, 미국에서는 파운드 기호 또는 파운드 키로도 많이 불린다.

HTML에서 'Anchor'로 사용되는 문자다.

; 쌍반점

U+003B


부호 ‘;’의 이름. 문장을 일단 끊었다가 이어서 설명을 더 계속할 경우에 쓴다. 주로 예를 들어 설명하거나 설명을 추가하여 덧붙이는 경우에 쓴다. ≒반구절점, 세미콜론.

세미 콜론. 문장(statement)의 끝을 의미한다.

: 쌍점

U+003A


문장 부호의 하나. ‘:’의 이름이다. 표제어 다음에 해당 항목을 들거나 설명을 붙일 때 쓰거나, 희곡 따위에서 대화 내용을 제시할 때, 시(時)와 분(分), 장(章)과 절(節) 따위를 구별할 때, 의존 명사 대(對)가 쓰일 자리에 쓰인다. ≒그침표, 쌍모점, 이중점, 콜론, 포갤점.

콜론. 프로토콜(http, https) 뒤에 반드시 적는다.

% 퍼센트

U+0025


백분율을 나타내는 단위. 기호는 %. ≒프로.

퍼센트는 URL에서 'escape' 문자로 사용되므로 안전하지 않은 문자다. 퍼센트 뒤에 아스키 코드의 16진수 값을 붙여 사용한다.
